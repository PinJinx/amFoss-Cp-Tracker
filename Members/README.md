# amFOSS CP Tracker

**Note:**
This is a special branch. There are no strict time limits, but it is **recommended to complete at least 5 problems per week**.
A final deadline for the entire task will be announced later.

👉 View NeetCode 150 Problems: [https://neetcode.io/practice/practice/neetcode150](https://neetcode.io/practice/practice/neetcode150)

---

## 📌 About

This repository is used to **track and manage the weekly Competitive Programming progress** of amFOSS members.

🏆 Live Leaderboard: **[Click Here!](https://am-foss-cp-tracker.vercel.app/)**

---

## 🧑‍💻 How to Participate

* Solve problems from the NeetCode 150 list
* Take a screenshot showing the problem is **Accepted**
* Upload the screenshot to your member folder via a PR
* Follow the detailed instructions provided in the folder
* Update your personal README with completed problems

---

## 📜 Rules

* Submit **only your own** accepted solutions
* Only **one submission per day per person**
---

## 💡 Tip

Consistency matters more than speed.
Even **5 problems per week = 150 in ~7 months**, which is a solid pace alongside academics.
